# Rick & Morty iOS App

An iOS application that displays characters and episode information from the Rick & Morty API.

## Features

- View characters list
- Character detail screen
- Network data fetching from API
- Responsive UI for iPhone and iPad

## Tech Stack
- Swift
- SwiftUI
- MVVM Architecture
- URLSession for networking
- Async/Await

## Requirements

- iOS 15.0+
- Xcode 14+
- Swift 5.7+

## Installation

1. Clone the repository
git clone git@github.com:sksandeepmca/Rickandmortydetailsdata.git

2. Open the project

cd RickAndMortyEpisode
open RickAndMortyEpisode.xcodeproj

Run the project in Xcode.

API Used

Rick and Morty API

https://rickandmortyapi.com/api/character

Project

Architecture

The project follows MVVM (Model View ViewModel) architecture.

Model → Data structures

View → UI components

ViewModel → Business logic and state management
